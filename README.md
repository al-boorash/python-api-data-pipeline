# Automated Data Pipeline (Python & API)
This project demonstrates a production-ready **ETL pipeline** that automates data ingestion from external REST APIs.

### Key Technical Features:
* **API Integration:** Fetches real-time JSON data using the `requests` library.
* **Security First:** Implemented `.gitignore` and `config.py` to protect sensitive API credentials—leveraging my **Cybersecurity** background.
* **Data Structuring:** Uses **Pandas** to transform raw JSON into a structured CSV format for business reporting.
